API Chess Documentation

Welcome to the API Chess Documentation! This guide will provide you with the details of the available API endpoints for managing chess games.

Base URL

All API requests should be made to the following base URL:

https://chess.tranducduyne14.repl.co/api

Endpoints

  • Move Piece on Chess Board

    Move a chess piece on the board.

    POST /move/:id
  • Player Role Assignment

    Assign a player's role (white or black) for a chess game.

    GET /player/:id
  • Chess Board Rendering

    Render a chess board for a game, suitable for new or ongoing games.

    POST /board/:id
  • Remove Chess Board

    Remove a chess board for a game.

    DELETE /board/remove/:id

Response Format

Responses from the API are returned in IMAGE format.

     { IMAGE }

In case of an error, the response format will be:

      {
        "status": false,
        "message": "Error message"
      }

Example Usage

To move a piece on the chess board:

      POST /api/move/12345
      Request Body:
      {
        "from": "e2",
        "to": "e4"
      }
      Response: 
      {
        "status": true,
        "play": 'w',
        "url": `https://chess.tranducduyne14.repl.co/test.png`
      }
      
In case of an error, the response format will be:

      {
        "success": false,
        "message": "Invalid move! Please try again.",
      }

To render a chess board for a game, suitable for new or ongoing games.:

      POST /api/board/12345
      Request Body: null
      Response: 
      {
        "status": true,
        "play": 'w',
        "url": `https://chess.tranducduyne14.repl.co/test.png`
      }
      
In case of an error, the response format will be:

      {
        "success": false,
        "message": "An error occurred while creating the chessboard!",
      }

For more information, please contact the api author.
